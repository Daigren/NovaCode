# Отчет по практической работе №4

### Цель работы:

1. Подобрать визуальные референсы для индивидуального проекта образовательной платформы NovaCode.
2. Определить и обосновать стиль будущего интерфейса (Tech Noir / Минималистичный киберпанк).
3. Утвердить цветовую палитру и настроить CSS-переменные проекта.
4. Разработать шрифтовую систему (типографику) на основе семейства шрифтов Inter.
5. Описать промежуточную структуру и эскизы ключевых интерфейсных окон (включая модуль умной ИИ-консультации).
6. Зафиксировать дизайн-концепцию в проектной документации и связать её с репозиторием.

---

### 1. Подбор и анализ визуальных референсов

Для формирования визуального языка NovaCode были проанализированы три современных интерфейса high-tech направленности:

1. **Vercel (vercel.com)**
* *Что заимствуется:* Идея глубокого абсолютно черного фона (`#000000`) в сочетании со строгой геометрической сеткой и тонкими контрастными границами блоков. Минималистичные карточки курсов без лишнего визуального шума.
* *Анализ:* Позволяет сфокусировать внимание пользователя на коде и учебном контенте, создавая ощущение премиальности цифрового продукта.


2. **Линейка интерфейсов Apple Developer / Эффекты macOS**
* *Что заимствуется:* Реализация навигационной панели (Header) с эффектом матового стекла (**Glassmorphism**) с использованием комбинации свойств `backdrop-filter: blur()` и полупрозрачной подложки.
* *Анализ:* Закрепленная (sticky) шапка сайта при прокрутке красиво размывает нижележащие элементы, сохраняя глубину интерфейса и не перегружая экран статичными монолитными плашками.


3. **Linear App (linear.app) / GitHub Next**
* *Что заимствуется:* Акцентное неоновое свечение элементов на темном фоне, использование мягких размытых фоновых кругов (glow effects) для выделения смысловых блоков, а также темная тема интерактивных терминалов.
* *Анализ:* Псевдоэлементы `::before` и `::after` с эффектом `filter: blur(150px)` добавляют интерфейсу динамики, превращая стандартный сайт в футуристичное рабочее пространство («Tech Noir»).

### 2. Визуальное направление проекта

Дизайн-концепция NovaCode определяется как **минималистичный технологичный нео-нуар (Tech Noir) с элементами киберпанка**.

* **Атмосфера:** Профессиональная, технологичная, футуристичная. Сайт должен ассоциироваться у молодых разработчиков с продвинутой средой разработки (IDE).
* **Геометрия:** Строгие прямоугольные формы со скруглением углов в диапазоне `5px` (для классических кнопок) до `12px–16px` (для окон терминала и чат-бота), что сохраняет баланс между строгостью кода и дружелюбностью интерфейса.
* **Интерактивность:** Плавные переходы (`transition: all 0.3s ease`), подсвечивание кнопок и инпутов неоновым свечением при наведении (hover) и фокусе (focus).

### 3. Цветовая палитра проекта

Цветовые решения жестко зафиксированы в глобальной области видимости CSS (`:root`) в файле `style.css`, что позволяет централизованно управлять темами оформления.

| Имя переменной | HEX / RGBA код | Роль в интерфейсе |
| --- | --- | --- |
| `--bg-color` | `#000000` | Основной фоновый цвет страниц |
| `--text-primary` | `#ffffff` | Цвет основного текста, заголовков h1 и ярких кнопок |
| `--text-secondary` | `#888888` | Цвет пояснительного текста, подписей и иконок |
| `--gray-800` | `#111111` | Фоновая подложка карточек, окон чата и полей ввода |
| `--gray-700` | `#333333` | Тонкие разделительные линии, рамки инпутов и блоков |
| `--neon-cyan` | `#06b6d4` | Главный акцентный неоновый цвет (светодиодный циан) |
| `--neon-cyan-dim` | `rgba(6, 182, 212, 0.15)` | Полупрозрачное свечение, фоновые круги, ховеры |

Дополнительно в блоках шапки используется адаптивное размытие с цветом `rgba(0, 0, 0, 0.76)` для сохранения читаемости поверх прокручиваемого текста.

### 4. Шрифтовая система (Типографика)

Основой текстовой экосистемы выбран современный гротеск **Inter**, оптимизированный для чтения программного кода и интерфейсных интерфейсов с экранов мониторов. Для декоративных крупных заголовков используется акцентный шрифт **Black Han Sans**.

* **Глобальные настройки (`body`):**
* `font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;`
* `line-height: 1.5;`


* **Иерархия размеров:**
* Главные заголовки блоков (`#mainContent h1`): `36px`, шрифт `Black Han Sans`.
* Заголовки разделов ИИ (`#ai-advisor`): `30px`, шрифт `Black Han Sans`.
* Пояснительные тексты / подзаголовки (`#description`): `25px`, начертание Regular.
* Текст в шапке сайта (`header`) и кнопках (`.consultation_2`): `22px`.
* Инпуты, текст сообщений в чате (`.message`): `16px` (`1rem` / `0.95rem`), начертание Regular/Medium.



Шрифты импортируются локально через конструкцию `@font-face` в начале файла `style.css`, предотвращая сбои при отсутствии интернета на стороне клиента.

### 5. Промежуточные эскизы основных экранов (Описание логики верстки)

#### Эскиз №1: Навигационная панель (Header)

* **Структура:** Логотип слева (`assets/images/icon.svg`), навигационные ссылки по центру («Главная», «Курсы», «Профиль»), интерактивная кнопка-переключатель «Консультация» справа.
* **Поведение:** `position: sticky; top: 0;` — закреплена вверху экрана. Кнопка консультации имеет прозрачный фон с белой рамкой, при наведении плавно инвертируется в чисто белый фон с черным текстом (`transition: all 0.5s ease`).

#### Эскиз №2: Главная страница / Промо-блок (`index.php`)

* **Контейнер `#mainContent`:** Центрированный блок с внешними отступами (`margin: 0px 460px;`), рамкой `#636363` и фоновой абстрактной текстурой `world.jpg`. Для сохранения темного тона применено свойство `background-blend-mode: multiply` с черной заливкой 60% непрозрачности.
* **Медиа-элемент (`#videoSection`):** Интегрированный HTML5-видеоплеер `<video>` без панели управления (`controls` отсутствуют), работающий в режиме бесконечного зацикленного фонового ролика со свойствами `autoplay muted loop playsinline`. Занимает ровно 50% ширины блока.

#### Эскиз №3: Интерфейс ИИ-Консультанта (`consultation.php`)

* **Окно терминала `.ai-chat-window`:** Центрированное окно шириной `max-width: 700px`. Задний фон — полупрозрачный графит `rgba(17, 17, 17, 0.6)` с эффектом размытия заднего плана `backdrop-filter: blur(10px)`.
* **Заголовок чата:** Снабжен зеленым неоновым индикатором статуса сети (`.neon-dot` со свойством `box-shadow: 0 0 10px var(--neon-cyan)`).
* **Поток сообщений (`.chat-messages`):** Высота `550px`, вертикальный скролл. Ответы ИИ выравниваются по левому краю, имеют подложку `--gray-800` и яркую неоновую вертикальную полосу слева (`border-left: 2px solid var(--neon-cyan)`). Сообщения пользователя прижаты к правому краю.
* **Строка ввода:** Нижнее поле ввода текста `#aiInput` полностью прозрачное, без рамок, с кнопкой отправки `.btn-send`.

### 7. Контрольные вопросы и ответы

1. **Какова основная цель данной практической работы?**
* Формирование и документальное закрепление целостной визуальной экосистемы веб-платформы (цвета, шрифты, стилистика блоков). Это необходимо для того, чтобы при дальнейшей верстке страниц сайт выглядел монолитно, а элементы управления не создавали визуального хаоса.


2. **Какие результаты должны быть получены после выполнения этапа?**
* Сформированный стилевой профиль (Tech Noir), настроенная таблица CSS-переменных, подключенные файлы шрифтов Inter, спроектированные макеты страниц (Главная и Консультация) и зафиксированная ссылка на прототип в Git.


3. **Как данный этап связан с предыдущими этапами проекта?**
* На предыдущих этапах была спроектирована структура баз данных (`db.php`) и функциональная схема страниц. Практическая работа №4 "надевает" на этот каркас визуальную оболочку, связывая HTML-блоки интерфейса с логикой асинхронных запросов JavaScript (например, отправка сообщений ИИ-агенту через API OpenRouter).


4. **Какие решения были приняты в рамках работы и почему?**
* Был выбран темный футуристичный стиль с неоновым цианом в качестве акцента. Это обусловлено целевой аудиторией (молодые программисты, привыкшие к темным темам в IDE). Для обеспечения безопасности API-ключ DeepSeek/OpenRouter был вынесен из публичной верстки в изолированный файл `includes/secrets.php`, добавленный в `.gitignore`.


5. **Какие файлы были добавили или изменили в Git-репозитории?**
* Добавлен файл стилей `assets/css/style.css`, скрипт логики чата `assets/js/ai-chat.js`, страница интерфейса `consultation.php`, скрытый файл конфигурации ключей `includes/secrets.php`, файл обработчика API `includes/api/ai-advisor.php` и файл документации `docs/figma-link.md`.


6. **Как результаты работы будут использоваться в следующих практических работах?**
* Сформированные стили и структура станут основой для создания полноценного UI-kit (компонентов кнопок, карточек курсов, модальных окон) и позволят быстро сверстать личный профиль студента (`profile.php`) и каталог языков программирования (`catalog.php`).
