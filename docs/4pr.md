# Практическая работа №4: Разработка дизайн-концепции и визуального направления проекта

**Проект:** NovaCode — Платформа для обучения программированию (Online & Offline)

## 1. Подбор референсов
Для разработки визуального стиля NovaCode были проанализированы следующие проекты:

1.  **Stripe (stipe.com)**
    *   *Что взято:* Чистота интерфейса, работа с типографикой и аккуратные тени. Использование градиентов и акцентных цветов для выделения технологичности.
2.  **Vercel (vercel.com)**
    *   *Что взято:* Строгий монохромный стиль (черный и белый), использование шрифтов без засечек (Geist/Inter) и четкое разделение блоков.
3.  **Refactoring.guru**
    *   *Что взято:* Принципы подачи образовательного контента, лаконичные карточки курсов и акцент на читаемости текста.

## 2. Описание визуального направления
Визуальная концепция NovaCode строится на сочетании **минимализма** и **технологичности (Tech-forward)**. 
- **Стиль:** Современный гротеск.
- **Акценты:** Использование "ломаных" форм (скошенные углы, динамичные блоки) для создания ощущения движения и прогресса.
- **Атмосфера:** Профессиональная, но доступная; интерфейс не перегружен лишними деталями, чтобы пользователь мог сосредоточиться на обучении.

## 3. Цветовая палитра
Выбрана высококонтрастная схема с одним ярким акцентным цветом:

*   **Primary Active:** `#4CD1CE` (Яркий бирюзовый) — используется для фонов секций, кнопок консультаций и акцентов. Символизирует энергию и современные технологии.
*   **Base Black:** `#000000` — основной цвет текста и ключевых декоративных элементов (разделители, иконки).
*   **Base White:** `#FFFFFF` — фон карточек и основного контента. Обеспечивает "воздух" в интерфейсе.
*   **Secondary Background:** `#D9ECEC` — мягкий оттенок для второстепенных плашек и категорий.

## 4. Шрифтовая система
Используется современный Sans-serif шрифт:

*   **Заголовки:** Inter / Outfit (Bold/Medium). Большой кегль (72px+) для главных заголовков, чтобы подчеркнуть уверенность бренда.
*   **Основной текст:** Inter (Regular/Medium). Межстрочное расстояние (leading) увеличено до 1.8 для лучшей читаемости длинных описаний курсов.
*   **Акценты:** Использование моноширинных шрифтов (JetBrains Mono) для элементов кода или технических идентификаторов.

## 5. Промежуточные эскизы (Wireframes/Drafts)
В рамках текущей реализации в коде уже подготовлены ключевые экраны:
- **Главная страница:** Секция с категориями (сетка 6x2) и блок "Самые популярные" с динамичным скошенным заголовоком.
- **Страница курсов:** Разделение на "Онлайн" (белый фон) и "Офлайн" (бирюзовый фон) курсы с крупными карточками описаний.
- **Модальное окно:** AI-консультант с чат-интерфейсом.
